**Problem 1:** Poprzednia karta musi CAŁKOWICIE znikać za następną (nie być częściowo widoczna).
```css
/* Każda karta stackuje się tak, że górna karta całkowicie przykrywa poprzednią */
.project-card {
  position: sticky;
  top: 80px;
  /* Karta ma pełną wysokość kontenera — przykrywa poprzednią w 100% */
}
/* Poprzednia karta skaluje się i dostaje scale(0.97) opacity(0) przy scrollu */
```
Implementacja GSAP:
```js
cards.forEach((card, i) => {
  if (i < cards.length - 1) {
    gsap.to(card, {
      scale: 0.94,
      opacity: 0,
      scrollTrigger: {
        trigger: cards[i + 1],
        start: 'top 80px',
        end: 'top 40px',
        scrub: true,
      }
    })
  }
})
```
